# Museum of Thought

Museum of Thought is a premium, atmospheric thought-mapping app built with Next.js.

## MVP features
- Add thoughts and sub-thoughts
- Interactive thought space with draggable orbs
- Thought detail panel
- Reflection mode
- Local save via localStorage
- AI analysis endpoint using OpenAI

## Stack
- Next.js
- React
- CSS
- localStorage
- OpenAI API

## How it works
You create thoughts.
Each thought becomes a node (orb) in space.

You can:
- connect thoughts
- explore relationships
- analyze meaning with AI
- reflect deeper using guided questions
